Phonic Frequency Distribution Parser

It a web application working on the concept of Audio Signal Processing that takes an audio input and returns an image consisting of its frequency components in a 2D plane. At this website user will be allowed to input any signal of the choice and after the Processing the signal at the backend, spectrun of the input signal will be displayed to the user as output.
